Antigravity Snippets

Uma coleção de snippets em CSS, HTML e JavaScript criada para o Google Antigravity. A proposta é oferecer soluções práticas e reutilizáveis que ajudam desenvolvedores a otimizar tempo, manter consistência no código e acelerar o processo de desenvolvimento. Cada snippet foi pensado para ser simples de integrar, fácil de adaptar e seguir boas práticas, garantindo desempenho e qualidade em qualquer projeto web.

Por que esse projeto existe?

Se você usa o Google Antigravity, sabe que ele não tem Settings Sync. Formata a máquina? Troca de computador? Lá se vão todos os seus snippets, keybindings e configurações do editor.

O antigravity-config nasceu da minha frustração com isso. Eu queria uma forma simples de carregar meus snippets e configs em qualquer setup novo — sem precisar reconfigurar tudo do zero, sem copiar arquivo por arquivo.

A solução foi colocar tudo num repositório Git: 49 snippets de CSS, HTML e JavaScript que eu uso no dia a dia, mais um script que restaura tudo com um único comando. Clona, roda o script, e pronto — seu editor tá configurado.

Decidi abrir o projeto porque pode ser útil pra mais gente. Mesmo que você não use todos os snippets, pode clonar o repo e adaptar à sua forma de trabalhar.
